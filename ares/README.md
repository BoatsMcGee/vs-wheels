# VapourSynth Ares

This package contains the [VS-AudioResample](https://github.com/ropagr/VS-AudioResample) VapourSynth plugin.

## Installation

```bash
pip install vapoursynth-ares
```

## Building from source

```powershell
uv build --package vapoursynth-ares
```
