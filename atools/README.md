# VapourSynth ATools

This package contains the [VS-AudioTools](https://github.com/ropagr/VS-AudioTools) VapourSynth plugin.

## Installation

```bash
pip install vapoursynth-atools
```

## Building from source

```powershell
uv build --package vapoursynth-atools
```
